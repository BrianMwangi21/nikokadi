# Niko Kadi

Stateless proof-of-participation badges for Kenya's voter-registration wave.

## What it does
- Upload a screenshot from `verify.iebc.or.ke`
- Read it with OCR
- Mint a signed QR badge
- Verify the badge on scan, with no database

## Run locally
```bash
npm install
npm run dev
```

Open `http://localhost:3000`.


Feel free to contribute.
